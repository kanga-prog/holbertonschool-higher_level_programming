Commande WC : La longueur de vos fichiers sera vérifiée à l'aide de wc.
Exigences en matière de documentation
Vous devez rédiger une documentation pour tout votre code. La documentation aide les autres (et vous-même) à comprendre le but et l'utilisation de votre code.

Modules : Chaque module (fichier Python) doit avoir une docstring expliquant son objectif.
Classes : Chaque classe doit avoir une docstring expliquant son objectif, ses attributs et ses méthodes.
Méthodes/Fonctions : Chaque méthode ou fonction doit avoir une docstring expliquant ce qu'elle fait, les paramètres qu'elle prend et ce qu'elle renvoie.
